# Cimarron 3D Printer Electronics

## Chipset

Mediatek MT7620N

## Other ICs Being Used

Trinamic TMC260 Stepper Driver

Winbond W9751G6KB-25 512MB DDR2 SDRAM

## Operating System

Embedded MIPS GNU/Linux OpenWRT

## Bootloader

DAS U-Boot
