# Cimarron 3D Printer Electronics

## Chipset

Octavo Systems/TI OSD335x

## Other ICs Being Used

Trinamic TMC2130 Stepper Driver

EMMC**G-S100 Embedded MMC

## Operating System

Embedded ARM GNU/Linux Kamikaze

## Bootloader

DAS U-Boot
